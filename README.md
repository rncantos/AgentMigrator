## 🚀 Features

- **Universal Translation:** Migrates agents across **Claude**, **Gemini**, and **Copilot** with 100% parity.
- **Rich Context Preservation:** Automatically maps custom metadata like `role`, `description`, `toneAndStyle`, and `examples` into the target system prompts.
- **Purist JSON Output:** Generates clean, production-ready payloads tailored specifically for each provider's API.
- **"WOW" User Interface:** A stunning, animated glassmorphism Webview built with modern aesthetics.
- **Secure by Design:** Built-in XSS prevention via Content Security Policy and large-file DoS protection.

## 🛠 Usage

1. Open the Command Palette (`Cmd+Shift+P` on Mac, `Ctrl+Shift+P` on Windows).
2. Type **`AgentMigrator: Migrate AI Agent`** and press Enter.
3. Select your **Origin AI** and **Target AI**.
4. Upload your agent's JSON configuration file.
5. Click **Migrate Agent 🚀** and choose where to save the converted file!
